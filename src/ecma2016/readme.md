## ECMAScript 2016

### Array.prototype.includes

`Array.prototype.includes(value : any) : boolean`

Examples:

```
    > ['a', 'b', 'c'].includes('a')
    true
    > ['a', 'b', 'c'].includes('d')
    false
```

### Exponentiation operator (幂运算符)

An infix operator for exponentiation

produces the same result as

    `Math.pow(x, y)`

Examples:
```
    let squared = 3 ** 2; // 9

    let num = 3;
    num **= 2;
    console.log(num); // 9

```
